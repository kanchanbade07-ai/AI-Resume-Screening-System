# 🤖 AI Resume Screening System

An AI-powered Resume Screening and Job Matching System built using **Spring Boot, Hibernate, MySQL, JWT Authentication, Apache PDFBox, and REST APIs**. The system automatically parses PDF resumes, extracts candidate information, ranks candidates based on skill matching, and provides recruiters with an interactive dashboard for efficient hiring.

---

## 🚀 Features

- 📄 Upload PDF resumes
- 🔍 Automatic resume parsing using Apache PDFBox
- 👤 Candidate information extraction
- ⭐ AI-based resume ranking
- 📊 Dashboard analytics
- 📝 Recruiter notes management
- 🔐 Secure JWT authentication
- 📥 Resume download
- 🗑️ Delete uploaded resumes
- 📋 Resume details view
- 🌐 RESTful API architecture

## ⚙️ Installation & Setup

### 1. Clone Repository

```bash
git clone https://github.com/kanchanbade07-ai/AI-Resume-Screening-System.git
```

### 2. Open Project

Import the project into **IntelliJ IDEA** or **Eclipse**.

### 3. Configure Database

Create a MySQL database and update the database credentials in:

```
src/main/resources/application.properties
```

### 4. Build Project

```bash
mvn clean install
```

### 5. Run Application

```bash
mvn spring-boot:run
```

Server starts at:

```
http://localhost:8080
```
