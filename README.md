# 🤖 AI Resume Screening System

An AI-powered Resume Screening and Job Matching System developed using **Spring Boot**, **Hibernate**, **MySQL**, **JWT Authentication**, and **Apache PDFBox**. The system automatically parses resumes, extracts candidate information, ranks resumes based on skills, and provides a modern dashboard for recruiters.

---

## 🚀 Features

- 📄 Resume Upload (PDF)
- 🔍 Automatic PDF Parsing
- 👤 Candidate Information Extraction
- ⭐ Resume Ranking based on Skills
- 📊 Dashboard Analytics
- 📝 Recruiter Notes Management
- 🔐 JWT Authentication
- 📥 Resume Download
- 🗑 Resume Delete
- 📋 Resume Details View
- REST API Architecture

## ⚙️ How to Run

### Clone Repository

```bash
git clone https://github.com/kanchanbade07-ai/AI-Resume-Screening-System.git
```

### Open Project

Import the project into IntelliJ IDEA or Eclipse.

### Configure Database

Create a MySQL database and update the database credentials in:

```
application.properties
```

### Run Application

Run the Spring Boot application.

Server starts at:

```
http://localhost:8080
```
